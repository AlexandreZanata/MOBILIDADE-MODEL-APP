# DriverBilling Baseline Checklist

- [ ] Carregamento inicial mostra spinner.
- [ ] Header com título e subtítulo aparece.
- [ ] Card de resumo mostra total pendente.
- [ ] Ação "Pagar Tudo" abre modal PIX quando há débito.
- [ ] Lista de ciclos renderiza todos os cards.
- [ ] Botão "Carregar mais" pagina corretamente.
- [ ] Botão "Gerar QR Code PIX" abre modal no ciclo elegível.
- [ ] Botão "Copiar Código PIX" copia texto e mostra feedback.
- [ ] Modal fecha no ícone de fechar e no back do sistema.

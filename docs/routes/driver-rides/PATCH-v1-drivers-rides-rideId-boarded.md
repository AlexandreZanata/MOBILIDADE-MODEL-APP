# PATCH /v1/drivers/rides/{rideId}/boarded

**Tag:** Driver - Rides  
**Summary:** Mark passenger boarded  
**Description:** Driver indicates that the passenger has boarded the vehicle.  
**Authentication:** Bearer token required (`Authorization: Bearer <accessToken>`).

---

## Parameters

| Name     | Location | Type   | Required | Description      |
|----------|----------|--------|----------|------------------|
| `rideId` | path     | string | Yes      | UUID of the ride |

---

## Request Body

None.

## Frontend Integration Notes

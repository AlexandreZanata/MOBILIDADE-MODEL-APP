# DELETE /v1/drivers/profile-photo

**Tag:** Drivers  
**Summary:** Remove profile photo  
**Description:** Removes the driver's profile photo.  
**Authentication:** Bearer token required (`Authorization: Bearer <accessToken>`).

---

## Parameters

None.

---

## Request Body

None.

## Frontend Integration Notes

- Called by `driversFacade.removeProfilePhoto()` via `useRemoveProfilePhoto` mutation
- On `onSuccess`, invalidate `driverProfileKeys.me()` to refresh the cached profile
- Always show a `ConfirmDialog` before calling this mutation
- On `401`, the facade layer triggers the token refresh flow automatically

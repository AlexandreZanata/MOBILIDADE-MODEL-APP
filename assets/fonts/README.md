# Fontes

Para instalar as fontes Poppins:

1. Baixe as fontes de https://fonts.google.com/specimen/Poppins
2. Instale as seguintes variações:
   - Poppins-Regular.ttf (weight 400)
   - Poppins-SemiBold.ttf (weight 600)
   - Poppins-Bold.ttf (weight 700)

Coloque os arquivos .ttf nesta pasta.

O aplicativo funcionará sem as fontes, usando as fontes do sistema como fallback.
